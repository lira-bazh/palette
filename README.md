# Моя палитра

Посетить: <a href="https://palette-orpin.vercel.app/">palette-orpin.vercel.app</a>

<img src="./screen.png">

Стэк: TypeScript, React, Redux Toolkit, Scss, CSS Modules, Vite

## О сервисе

Сервис позволяет сохранить палитру цветов

* Цвета можно вводить в hex, rgb или hsv формате.
* Можно сохранить название палитры.
* В палитре может быть минимум один нет.
* Палитры хранятся локально в браузере.
* К каждому цвету можно добавить комментарий.
* Порядок цветов можно менять, перетаскивая их в таблице.
* Сервис поддерживает тёмную и светлую тему.

## Как запустить?
    npm install
    npm run dev
Запускается по адресу http://localhost:5173
